## Цель
На мобильных устройствах секция «Что мы строим» должна быть короче — карточки услуг идут по 2 в ряд (вместо 1), с уменьшенными отступами.

## Что меняем
Файл: `src/components/site/sections.tsx`, компонент `Services`.

### 1. Сетка карточек
Сейчас:
```
grid gap-6 sm:grid-cols-2 lg:grid-cols-3
```
Станет:
```
grid grid-cols-2 gap-3 sm:gap-6 lg:grid-cols-3
```
Эффект: уже с 390px ширины будет 2 колонки, отступы между карточками меньше на мобиле.

### 2. Компактные карточки на мобиле
Внутри карточки:
- padding: `p-7` → `p-4 md:p-7`
- иконка: `h-12 w-12` → `h-9 w-9 md:h-12 md:w-12`
- заголовок `h3`: `mt-5 text-xl` → `mt-3 text-base md:mt-5 md:text-xl`
- описание `p`: `mt-2 text-sm` → `mt-1 text-xs md:mt-2 md:text-sm`
- ссылка «Подробнее»: `mt-5 text-sm` → `mt-3 text-xs md:mt-5 md:text-sm`
- верхняя градиентная полоска `h-1` остаётся.

### 3. Отступы секции и заголовка
- Секция: `py-20 md:py-28` → `py-12 md:py-28`
- Сетка сверху: `mt-12` → `mt-8 md:mt-12`
- Заголовок `h2`: добавим `text-2xl md:text-5xl` (вместо `text-3xl … md:text-5xl`) — чуть меньше на мобиле.

## Что НЕ трогаем
- Десктопный вид (≥ md) остаётся прежним.
- Содержимое карточек, иконки, переводы, ссылки — без изменений.
- Другие секции — без изменений.
