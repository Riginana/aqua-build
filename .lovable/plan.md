
## Сделать кнопку «Смотреть проекты» тихой текстовой ссылкой

В Hero на главной кнопка «Смотреть проекты» сейчас оформлена как заметная outline-кнопка с белой рамкой. Превратим её в скромную текстовую ссылку рядом с основным CTA, чтобы не конкурировать с «Бесплатная консультация».

### Изменение в `src/components/site/sections.tsx` (компонент `Hero`)

Заменить текущий `<Link to="/projects">` со стилями `border-2 border-white/80 ... rounded-full ...` на лаконичную текстовую ссылку:

- Без рамки, без фона, без скругления-пилюли.
- Светлый полупрозрачный текст (`text-white/80`), подчёркивание появляется при наведении (`hover:text-white hover:underline underline-offset-4`).
- Сохранить `min-h-[48px]` и `inline-flex items-center` для удобного тапа на мобильном.
- Оставить стрелку `→` после текста для лёгкого визуального акцента.

Основная кнопка «Бесплатная консультация» остаётся без изменений — она единственный яркий CTA в hero.

### Результат

В hero одна выразительная cyan-кнопка + спокойная белая текстовая ссылка «Смотреть проекты →». Визуальная иерархия становится чище, главный призыв к действию выделяется сильнее.
