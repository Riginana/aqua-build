## Цель
Сейчас 4 элемента в TrustBar (под Hero) на десктопе используют сетку из 5 колонок (`md:grid-cols-5`), из-за чего одна колонка пустует и элементы расположены неравномерно. Нужно распределить их равномерно.

## Что меняем
Файл: `src/components/site/sections.tsx`, компонент `TrustBar`.

### Изменение сетки
Сейчас:
```
grid grid-cols-2 gap-6 ... md:grid-cols-5
```
Станет:
```
grid grid-cols-2 gap-6 ... md:grid-cols-4
```

Так все 4 элемента (Опыт, География, Гарантия, Лидер) равномерно займут всю ширину контейнера на десктопе. На мобиле останется 2 колонки — без изменений.

## Что НЕ трогаем
- Сами элементы, иконки, тексты переводов.
- Мобильную раскладку (2 колонки).
- Остальные секции страницы.
