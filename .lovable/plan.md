## Объединить первые два фото в одну карточку

Первое и второе фото (`r1` и `r2`) — один и тот же объект (крытый бассейн с синей мозаикой, гидромассажем и лестницей). Объединю их в одну карточку с галереей из двух изображений.

### Изменения

**`src/components/site/projects-data.ts`**
- Расширить тип `Project`: добавить опциональное поле `images?: string[]` для карточек с несколькими фото. Поле `src` остаётся главным/обложкой.
- Объединить записи `r1` и `r2` в одну (id `r1`):
  - `src: r1` (обложка)
  - `images: [r1, r2]`
  - Заголовок: «Бассейн с гидромассажем» (RU) / «Гидромассаж бассейн» (KG)
  - Описание дополнить деталями со второй карточки (нержавеющая лестница, глубина 1,5 м)
- Удалить отдельную запись `r2`. Импорт `r2` оставить (используется в массиве `images`).
- Итог: на странице `/projects` будет 18 карточек вместо 19.

**`src/routes/projects.tsx`**
- В лайтбоксе (модалке) при открытии карточки с `images` показывать миниатюры/переключатель между фото объекта. Если `images` нет — поведение прежнее (одно фото).
- На самой карточке в сетке — без изменений (показывает `src` как обложку), опционально маленький бейдж «2 фото» в углу.

### Что НЕ меняется

- Остальные 17 карточек, фильтры, переводы, структура страницы.
- Главное hero-изображение.
